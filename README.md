# OverlayTimer in macOS

Маленька панелька поверх всіх вікон з годинником і таймером.
Написана на C++ і Objective-C++ без зовнішніх бібліотек.

## Можливості
- Годинник — показує поточний час
- Таймер — секундомір з кнопками старт/стоп/скинути
- Завжди поверх всіх вікон навіть повноекранних
- Можна перетягувати по екрану

## Встановлення

1. Клонуй репозиторій
git clone https://github.com/Alex-cpu343/OverlayTimer-in-macOs.git

2. Відкрий в Xcode
Двічі клікни на файл OverlayTimer.xcodeproj

3. Збери проєкт
Cmd + R щоб запустити або Product → Archive щоб зібрати .app

4. Додай в Applications
Перетягни OverlayTimer.app в папку Програми

## Іконка (за бажанням)
1. Знайди OverlayTimer.app в Finder
2. Права кнопка → Отримати інформацію
3. Перетягни свій PNG на іконку у лівому верхньому куті

# OverlayTimer in macOS

A lightweight always-on-top overlay bar with a clock and timer.
Built with C++ and Objective-C++ using no external libraries.

## Features
- Clock — shows current time
- Timer — stopwatch with start/stop/reset buttons
- Always on top of all windows including fullscreen
- Draggable anywhere on screen

## Installation

1. Clone the repository
git clone https://github.com/Alex-cpu343/OverlayTimer-in-macOs.git

2. Open in Xcode
Double click OverlayTimer.xcodeproj

3. Build the project
Press Cmd + R to run or go to Product → Archive to build .app

4. Add to Applications
Drag OverlayTimer.app to your Applications folder

## Custom Icon (optional)
1. Find OverlayTimer.app in Finder
2. Right click → Get Info
3. Drag your PNG onto the icon in the top left corner
